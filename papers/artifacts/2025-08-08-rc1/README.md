# GAT-FedPPO 实验工件 - Release 2025-08-08-rc1

## 概述

本目录包含GAT-FedPPO联邦学习系统的实验评测工件，对应论文中的性能评估部分。

## 评测协议

### 脚本版本
- **nightly_ci.py**: 2025-08-07-r4 (Wilson下界复评路径启用)
- **consistency_test_fixed.py**: 2025-08-07-r4 (PyTorch 2.6兼容)
- **monitoring_dashboard.py**: 最新版本

### 评测参数
- **样本量**: 400样本 (日间快检), 800样本 (夜间大测)
- **随机种子**: 42, 123, 2025 (三种子交叉验证)
- **Wilson下界**: 95%置信区间，安全边界3-4pp
- **复评机制**: 不稳定结果自动触发K=100二次验证

## 实验结果

### 微调前基线 (2025-08-07)
- **gulfport**: 标准49.0%(边缘), 完整35.2%
- **baton_rouge**: 中级43.5%, 高级31-38%
- **new_orleans**: 中级44-46%, 高级35-37%

### 微调后性能 (2025-08-08)
- **gulfport**: 标准53-55%(+4-6pp), 完整39-41%(+4-6pp)
- **baton_rouge**: 全部通过 (0 alerts)
- **new_orleans**: 中级48.5%(+3-4pp), 高级37.5%(+0.5-2.5pp)

### 技术改进
1. **针对性微调**: fine_tune_stages.py (3-8pp目标提升)
2. **模型格式统一**: checkpoint格式标准化
3. **兼容性修复**: PyTorch 2.6 weights_only参数
4. **监控优化**: 4pp安全边界减少误报

## 统计显著性

所有结果基于Wilson置信区间(95%)，确保统计显著性：
- **样本量**: 400-800样本确保足够统计功效
- **多种子验证**: 3个独立种子减少随机性影响
- **复评机制**: 不稳定结果自动二次验证
- **安全边界**: 3-4pp边界容忍自然方差
